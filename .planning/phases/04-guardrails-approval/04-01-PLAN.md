Create `minilegion/core/scope_lock.py`:

1. Import `sys` from stdlib, `ValidationError` from `minilegion.core.exceptions`.

2. Implement `normalize_path(path: str) -> str`:
   - Remove "./" prefix: `if path.startswith("./"): path = path[2:]`
   - Also handle ".\" prefix: `if path.startswith(".\\"): path = path[2:]`
   - Strip trailing slashes: `path = path.rstrip("/\\")`
   - Normalize backslashes to forward slashes: `path = path.replace("\\", "/")`
   - Lowercase on Windows: `if sys.platform == "win32": path = path.lower()`
   - Return path.
   - DO NOT use `os.path.normpath()` — it converts to backslashes on Windows.

3. Implement `check_scope(changed_files: list[str], allowed_files: list[str]) -> list[str]`:
   - Build `allowed_set = {normalize_path(f) for f in allowed_files}`
   - Return `[f for f in changed_files if normalize_path(f) not in allowed_set]`
   - Returns list of out-of-scope files from the ORIGINAL (un-normalized) `changed_files` values.

4. Implement `validate_scope(changed_files: list[str], allowed_files: list[str]) -> None`:
   - Call `check_scope()`. If result is non-empty, `raise ValidationError(f"Out-of-scope files: {out_of_scope}")`.
