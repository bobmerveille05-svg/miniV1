<objective>
Define Pydantic models for all 6 machine-readable artifact types, build a central schema registry, and pre-generate JSON Schema files.

Purpose: This is the validation foundation — every LLM response in the pipeline (Phases 3-10) will be validated against these schemas via the registry's `validate()` function. JSON Schema files enable external tool consumption.

Output: 5 new Pydantic models in schemas.py, registry module with validate/get_schema/get_json_schema, 6 pre-generated .schema.json files, generation script, and comprehensive tests.
</objective>
