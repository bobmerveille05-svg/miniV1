Create `minilegion/core/context_assembler.py` with a single public function:

```python
def assemble_context(tool: str, project_dir: Path, config: MiniLegionConfig) -> str:
    """Assemble a portable context block for the given AI tool.

    Reads: STATE.json (current stage, approvals, last history entries),
           adapter file at project-ai/adapters/<tool>.md (if present â€” optional),
           memory files at project-ai/memory/*.md (if present â€” optional),
           previous artifact summary (most recent *.md artifact in project-ai/),
           stage template at project-ai/templates/<stage>.md (if present â€” optional).

    Returns a markdown string with sections:
      ## Current State
      ## Previous Artifact
      ## Stage Template
      ## Memory
      ## Adapter Instructions
    """
```

**Section assembly rules:**
1. **Current State**: Read `STATE.json` via `load_state()`. Include `current_stage`, `completed_tasks` count, last 3 `history` entries (timestamp + action + details). Apply `config.context.max_injection_tokens` as a soft character limit (warn if exceeded â€” print to stderr, do NOT truncate silently).
2. **Previous Artifact**: Look for the most recently modified `.md` file in `project-ai/` matching `BRIEF.md`, `RESEARCH.md`, `DESIGN.md`, `PLAN.md`, `EXECUTION_LOG.md`, `REVIEW.md` based on `current_stage`. Read it. Truncate at `config.context.max_injection_tokens` chars with a `[TRUNCATED]` marker if needed.
3. **Stage Template**: Read `project-ai/templates/<current_stage>.md` if it exists. Otherwise, use a minimal stub: `_No template defined for stage {stage}._`
4. **Memory**: Concatenate `project-ai/memory/decisions.md`, `glossary.md`, `constraints.md` if any exist. Skip missing files silently.
5. **Adapter Instructions**: Read `project-ai/adapters/<tool>.md` if exists, else `project-ai/adapters/_base.md` if exists, else use a short default: `_Paste this context block at the start of your conversation with {tool}._`

Return the fully assembled markdown string. Do NOT write any files â€” that is the CLI command's responsibility.

Command logic:

5. `typer.echo(block)` â€” print to stdout

On `MiniLegionError`, print red error and exit 1 (same pattern as all other commands).
