    - Validates transition to the command's target stage
    - If valid: prints "Would run [stage]... (not yet implemented)" in green
    - If invalid: catches InvalidTransitionError, prints red error with stage names
    - plan accepts --fast (bool) and --skip-research-design (bool) flags
    - execute accepts --task (int|None) and --dry-run (bool) flags
    - brief accepts optional text argument for brief content

    No-args behavior:
    - `python run.py` with no command shows help listing all 8 commands

    Verbose flag:
    - `python run.py --verbose init myproject` sets verbose state and shows extra details

    Error handling:
    - All commands wrap operations in try/except for MiniLegionError subclasses
    - Errors displayed in red via typer.echo with typer.style
    - Exits with code 1 on error
  </behavior>
