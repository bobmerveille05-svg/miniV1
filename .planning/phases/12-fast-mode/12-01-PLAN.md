# Plan 12-01: Preflight Skip-Stages Support

**Phase:** 12 — Fast Mode
**Wave:** 1
**Requirements:** FAST-03 (partial), GUARD-01, GUARD-02

## Goal

Add `skip_stages` parameter to `check_preflight()` so that file/approval requirements for explicitly-skipped stages can be bypassed. This is the foundation for fast mode — downstream commands (execute, review, archive) call `check_preflight()` and must not fail when research/design artifacts are absent.

## Tasks

### Task 1: Add `STAGE_ARTIFACTS` dict to `preflight.py`

In `minilegion/core/preflight.py`, add a new mapping after `REQUIRED_APPROVALS`:

```python
# Artifacts produced by each stage (used to filter skipped-stage requirements).
STAGE_ARTIFACTS: dict[str, list[str]] = {
    "research": ["RESEARCH.json", "RESEARCH.md"],
    "design": ["DESIGN.json", "DESIGN.md"],
}

# Approvals set by each stage
STAGE_APPROVALS: dict[str, list[str]] = {
    "research": ["research_approved"],
    "design": ["design_approved"],
}
```

### Task 2: Update `check_preflight()` signature and logic

Change function signature from:
```python
def check_preflight(stage: Stage | str, project_dir: Path) -> None:
```
to:
```python
def check_preflight(
    stage: Stage | str,
    project_dir: Path,
    skip_stages: set[str] | None = None,
) -> None:
```

In the file-check loop, filter out files belonging to skipped stages:
```python
# Build the set of skippable filenames
skipped_files: set[str] = set()
if skip_stages:
    for s in skip_stages:
        skipped_files.update(STAGE_ARTIFACTS.get(s, []))

for filename in REQUIRED_FILES.get(stage, []):
    if filename in skipped_files:
        continue  # skip — stage was explicitly skipped
    if not (project_dir / filename).exists():
        raise PreflightError(f"Missing required file: {filename}")
```

In the approval-check block, filter out approvals belonging to skipped stages:
```python
skipped_approvals: set[str] = set()
if skip_stages:
    for s in skip_stages:
        skipped_approvals.update(STAGE_APPROVALS.get(s, []))

for approval_key in required_approvals:
    if approval_key in skipped_approvals:
        continue
    if not state.approvals.get(approval_key, False):
        raise PreflightError(f"Missing required approval: {approval_key}")
```
